Project Overview

This Airline Management System is designed to streamline the management of airline operations, focusing on staff, aircraft, and journey logs. It provides a user-friendly interface for managing staff records, including details like designation and leave entitlements. The system also allows for the addition, editing, and deletion of aircraft details, as well as maintaining journey logs for tracking flight operations. With features for searching and updating records, the system is built to improve efficiency and organization within airline management.
